Text Formatting untuk Texs dari Komputer

Perintah javascript untuk menampilakan teks console adalah console.log(). Kita juga bisa menampilkan isi variabel dengan fungsi ini. Misalkan kita punya variabel name, maka kode programnya bisa ditulis seperti ini: 

        var name = "Kelas X PPLG 2 ";
        console.log(name);
Maka hasil output: Kelas X PPLG 2

Untuk menjalankan ulang program, lakukan refresh dengan menekan tombol F5
